# 📋 Kanban Semanal

Un tablero Kanban personal para gestionar tareas semana a semana. Sin servidores, sin cuentas, sin complicaciones — corre directo en el navegador.

## ¿Qué hace?

- Tablero de 7 columnas, una por día de la semana (Lunes → Domingo)
- Resalta automáticamente el día de hoy
- Muestra la fecha exacta de cada día según la semana actual
- Barra de progreso con porcentaje de tareas completadas
- Las tareas se guardan en `localStorage` — persisten aunque cierres el navegador
- Botón **Nueva Semana** para limpiar el tablero cada lunes

## Funcionalidades

- ✅ Agregar tareas con `Enter` o el botón
- ✅ Marcar tareas como completadas con un click
- ✅ Eliminar tareas individualmente
- ✅ Contador de tareas completadas / total
- ✅ Confirmación antes de reiniciar la semana
- ✅ Diseño responsivo (funciona en móvil)

## Stack

HTML · CSS · JavaScript vanilla — cero dependencias, cero build.

## Uso

Abre `index.html` en cualquier navegador. No necesita servidor.
